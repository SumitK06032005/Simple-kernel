A simple 32-bit kernel that runs on x86 architecture and performs the following simple tasks:

- Echo - display whatever you typed for it to display  
- Change text color - to change the color of the entire text on screen from given choices  
- Shutdown - to shutdown the kernel  
- Kernel panic mode - when something goes wrong the message is displayed on screen
